# 道家八字排盘 - Bazi Astrology Analyzer

一个专业的中文八字排盘应用，采用道家/易学风格设计，展现传统智慧的现代呈现。

## 功能特性

✨ **核心功能**
- 输入生日年月日时间（支持时间精确到分钟）
- 自动生成详细的八字排盘分析
- 展示四柱八字（年、月、日、时柱）
- 分析五行分布和相生相克关系
- 计算纳音五行和十神解释
- 显示起运时间和大运分析
- 提供流年分析预测

🎨 **设计特点**
- 中国道家风格视觉设计
- 优雅的渐变色和金色装饰元素
- 流畅的动画和过渡效果
- 响应式设计，适配各种设备
- 传统中文排版和书法风格字体

## 技术栈

- **框架**: Next.js 14 + TypeScript
- **样式**: Tailwind CSS + Custom CSS
- **API**: 连接 Bazi 命理分析后端服务
- **字体**: Google Fonts - Noto Serif SC（思源宋体）

## 常见问题

**Q: 为什么需要输入具体的出生时间？**
A: 时柱直接影响到整个八字的五行分布和十神含义，精确的时间能提供更准确的分析。

**Q: 农历和阳历有什么区别？**
A: 八字排盘基于农历（阴历），系统会自动将阳历日期转换为农历。

**Q: 起运时间是什么意思？**
A: 指的是大运开始计算的时间点，一般为出生后若干年月日。

**Q: 流年是什么？**
A: 流年指当年运势，按照干支纪年周期循环，用来预测该年吉凶。
